Откройте index.html и введите имя пользователя, чьими фигурами будете управлять. Три квадрата справа принадлежат пользователю Other User, остальные — Igor Zalutsky. Фигуры перемещаются перетаскиванием. Двойной клик изменяет цвет квадрата на случайный. Для того, чтобы удалить квадрат, перетащите его на блок TRASH. Для создания нового нажмите кнопку "Create new square". Для удаления всех своих квадратов нажмите кнопку "Remove all my squares".

В процессе обнаружилась ошибка в серверном методе remove-shapes.do — он удаляет квадраты всех пользователей, а не только указанного в параметре userId. Поэтому удаление всех квадратов пришлось реализовать отдельными запросами поштучно.

Приложение построено на основе MVC-архитектуры. Зависимость модели от представления устранена при помощи событий — все свойства моделей являются ObservableProperties, на изменения которых подписаны представления. Модели хранятся в ObservableCollection.

Изменения реального времени эмулируются при помощи jsonp-поллинга с динамическим таймаутом, который автоматически подстраивается под "отзывчивость" сервера, чтобы избежать перегрузок при большом количестве клиентов.

Для работа с DOM используется jQuery, для drag-n-drop — jQuery UI, для корректной работы с функциями ECMA Script 5 — es5-shim.

Приложение протестировано в Chrome 21 под Mac OS X 10.6 и Firefox 14 под Windows 7.

Для сборки проекта использовался Grunt, для юнит-тестирования — QUnit, для проверки качества кода — JSHint, для минимизации — UglifyJS. Однако, в итоговом варианте подключены несжатые версии скриптов для большей наглядности.
